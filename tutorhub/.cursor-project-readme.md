# TutorHub

A two-sided tutor booking platform, built with Rails 7. See [`TutorHub-Sprint-Plan.md`](../../../TutorHub-Sprint-Plan.md) (original plan) and [`TASKS.md`](TASKS.md) (per-task estimate breakdown).

## Quickstart (after installing Ruby)

```bash
bundle install
bin/rails db:create db:migrate db:seed
bin/rails server
```

Tests: `bundle exec rspec`
Lint: `bundle exec rubocop`
Everything in Docker: `docker compose up`

## Layout

```
tutorhub/
├── app/                       # MVC + services
├── bin/                       # rails, bundle, rspec, rubocop scripts
├── config/                    # DB, routes, envs, initializers
├── db/                        # migrations, seeds, schema.rb
├── docs/                      # ARCHITECTURE, SETUP, GIT_FLOW
├── spec/                      # RSpec tests
├── .github/workflows/ci.yml   # CI
├── Dockerfile                 # Production image
├── docker-compose.yml         # One-command local stack
├── .rubocop.yml               # Lint config
└── TASKS.md                   # Sprint / hour estimates (JD requirement)
```
